РГР по Программированию
Группа: АВТ-019
Вариант: 14
Задача: Заменить все цифры на заданный символ

Для передачи данных используются UDP сокеты (локал хост)
Логи пишутся в log.txt
Обработанные файлы получают название <filename>-with-<char>.txt

Сборка:
make all

Для управления сервером (myserv.service) используется программа systemctl с соответствующими опциями (start, stop, restart, status)

Клиент:
./client <char> <filename[, ...]>
